HobbyGuide Bot
Telegram бот для управления кружками и записями на занятия

 О проекте
HobbyGuide - это многофункциональный Telegram бот для записи на различные кружки и занятия по интересам. Бот предоставляет удобный интерфейс для клиентов и администраторов.

 Основные функции
Для клиентов:
 Просмотр каталога кружков по категориям (танцы, рисование, пилатес, йога, спорт)

 Запись на занятия с выбором даты и времени

 Просмотр и управление своими записями

 Поддержка и FAQ

 Просмотр акций и специальных предложений

 Рекомендации товаров для кружков

 Оставление отзывов

Для администраторов:
 Добавление и управление кружками

 Управление акциями и скидками

 Обработка заявок на записи

 Ответы на вопросы поддержки

 Управление отзывами

Технологии

Logging - для логирования работы бота

 Возможные проблемы и решения
Проблема 1: Бот не запускается
Решение:

Проверьте установку зависимостей: pip list | grep telebot

Проверьте токен бота

Убедитесь, что Python версии 3.8+

Проблема 2: Ошибка подключения к БД
Решение:

Проверьте настройки подключения в db_config

Убедитесь, что MySQL сервер запущен: sudo service mysql status

Проверьте права пользователя базы данных

Проблема 3: Бот не отвечает на команды
Решение:

Проверьте, что бот запущен

Убедитесь, что используется правильный токен

Проверьте интернет-соединение

Проблема 4: Ошибки при бронировании
Решение:

Проверьте заполнение таблиц в БД

Убедитесь в корректности данных кружков

Проверьте логи на наличие ошибок
